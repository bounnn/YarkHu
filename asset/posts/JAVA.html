ຮູ້ບໍ່ ວ່າພາສາ JAVA ມີຄວາມເປັນມາແນວໃດ? ແລະມີຂໍ້ດີຂໍ້ເສຍແນວໃດ?

JAVA ເປັນພາສາທີ່ຂຽນໂປຣແກຣມເພື່ອວັດຖຸປະສົງທົ່ວໄປໂດຍສາມາດເຮັດວຽກໄປພ້ອມກັນເປັນພາສາທີ່ສ້າງມາຈາກ Class ແລະ ສະໜັບສະໝຸນການຂຽນໂປຣແຣມແບບ Object ຖືກອອກແບບມາໃຫ້ພ້ອມສໍາລັບການໃຊ້ງານໂດຍມີ Method ແລະ Class ຊ່ວຍອໍານວຍຄວາມສະດວກຢ່າງຫຼວງຫຼາຍພາສາ JAVA ນັ້ນມີຄວາມຕັ້ງໃນວ່າຈະເຮັດໃຫ້ນັກພັດທະນາອອກແບບ ແລະ ພັດທະນາໂປຣແກຣມໜ້ອຍລົງ ນັ້ນຄືການຂຽນພຽງຄັ້ງດຽວແຕ່ນໍາໄປໃຊ້ງານໄດ້ທຸກທີ່ ຫຼື ທຸກແພັດຟອມ.

ຕົ້ນກໍາເນີດການອ້າງອີງໃນການພັດທະນາຂອງ JAVA Compiler Virtual Machines ແລະ ຄາສໄລບຣາລີ ໂດຍຖືກແຜ່ຫຼາຍໂດຍ Sun ພາຍໃຕ້ລິຂະສິດທີ່ເໝາະສົມໃນເດືອນ 7 ປີ 2007 ໃນການປະຕິບັດຕາມກັບຂໍ້ລະບຸຂອງ JAVA Community Process Sun ໄດ້ຈົດທະບຽນໃໝ່ໃນເທັກໂນໂລຊີຂອງ JAVA ເປັນຈໍານວນຫຼາຍພາຍໃຕ້ GNU General Public License ແລະ ການພັດທະນາເທັກໂນໂລຊີ້ອື່ນໆ ເວີຊັ້ນລ້າສຸດຂອງ JAVA ຄື ເວີຊັ້ນ 8 ທີ່ສະໝັບສະໜຸນໂດຍ Oracle.

ແອບພິເຄຊັ້ນຂອງ JAVA ນັ້ນໂດຍປົກກະຕິແລ້ວຈະຄອມພາຍເປັນ Bytecode ທີ່ສາມາດລັນໄດ້ໃນ JAVA virtual machine ( GVM ) ປະຈຸບັນຖືກຊື້ກິດຈະການເປັນ Oracle Corporation ແລະ ເປີດຕົວຄັ້ງແລກເມື່ອປີ 1995 ໂດຍພາສານັ້ນໄດ້ຮັບຮູບແບບຈາກພາສາ C ແລະ C++ ແຕ່ພາສາ JAVA ຖືວ່າເປັນພາສາລະດັບສູງກວ່າພາສາທັງສອງ.

ຂໍ້ດີ ແລະ ຂ້ເສຍຂອງພາສາ JAVA

1. ຂໍ້ດີ

  • ສາມາດໃຊ້ໄດ້ທຸກແພດຟອມບໍ່ຈຳເປັນຕ້ອງແກ້ໄຂດັດແປງໃດໆເຊັ່ນ ເຮົາຂຽນໂປຣແກມໃຊ້ PC ທົ່ວໄປແຕ່ກໍສາມາດນຳໄປໃຊ້ໄດ້ໃນແພດຟອມອື່ນໆໄດ້ເຊັ່ນ:ໃນໂທລະສັບ.
  • ເປັນພາສາທີ່ຮອງຮັບການຂຽນເຊີງວັດຖຸແບບສົມບູນ ຊື່ງສາມາດຂຽນງານທີ່ຊັບຊ້ອນໄດ້ດີ.
  • ພາສາຈາວາມີຄວາມຊັບຊ້ອນທີ່ນ້ອຍກວ່າພາສາຊີໃຊ້ code ທີ່ນ້ອຍກວ່າແຕ່ສາມາດເຮັດວຽກໄດ້ຄືກັນ ຊື່ງຈະຊ່ວຍຫຼຸດຂໍ້ຄວາມຜິດພາດໃນການຂຽນໂປຣແກມ.
  • ແຈ້ງຂໍ້ຜິດພາດທັ້ງຕອນ compile ແລະຊ່ວງເວລາໃຊ້ໂປຣແກມ (Runtime) ຊື່ງເຮັດໃຫ້ການແກ້ໄຂ Bug ນັ້ນເຮັດໄດ້ງ່າຍກ່ວາການຂຽນພາສາອື່ນໆ.

2. ຂໍ້ເສຍ

  • ພາສາຈາວາຈະເຮັດວຽກໄດ້ຊ້າກວ່າພາສາໃນແບບ native code ເນື່ອງຈາກ code ຈາວານັ້ນຈະແປງເປັນພາສາກາງກ່ອນຈື່ງເຮັດໃຫ້ຂັ້ນຕອນຈະຫຼາຍກວ່າພາສາອື່ນໆ ທີ່ສາມາດແປງ code ໄປເປັນພາສາເຄື່ອໄດ້ເລີຍ.

ເຄື່ອງມືທີ່ມັກໃຊ້ໃນການຂຽນພາສາຈາວາ

ເຮົາສາມາດໃຊ້ Notepad ທົ່ວໄປ ຫຼື Edit plus ໃນການຂຽນກໍໄດ້ພຽງແຕ່ດາວໂຫຼວດ JDK ຈາກ Sun Microsystem ສໍາລັບໃນການ Compile Code ຊື່ງສາມາດດາວໂຫຼວດ JDK ໄດ້ເລີຍ.

ແຕ່ແນະນຳເຄື່ອງນິຍົມສໍາລັບຄົນທີ່ສົນໃຈຈະເລືອກພັດທະນາໂປຣແກມດ້ວຍພາສາຈາວາໂດຍຈະຮຽນເຄື່ອງມືເຫຼົ່ານີ້ IDE (Integrated Development Environment) ຊີ່ງມີໃຫ້ເລືອກໃຊ້ກັນຫຼາຍຕົວ ແຕ່ທີ່ແນະນຳຈະເປັນ IDE ທີ່ໃຊ້ກັນແພ່ຫຼາຍທົ່ວໂລກ.
