HTML

HTML ແມ່ນພາສາຫຼັກທີ່ໃຊ້ໃນການຂຽນໜ້າເວັບ ໂດຍໃຊ້ Tag ໃນການກໍານົດການສະແດງຜົນ HTML ຫຍໍ້ມາຈາກຄໍາວ່າ Hypertext Markup Language ໂດຍ Hypertext ໝາຍເຖິງຂໍ້ຄວາມທີ່ເຊື່ອມຕໍ່ກັນຜ່ານລິ້ງ ( Hyperlink ) Markup Language ໝາຍເຖິງພາສາທີ່ໃຊ້ Tag ໃນການກໍານົດການສະແດງຜົນໜ້າເວັບທີ່ຕ່າງກໍເຊື່ອມເຖິງກັນໃນ Hyperspace ຜ່ານ Hyperlink ນັ້ນເອງ.

ຄວາມເປັນມາຂອງ HTML ເລີ້ມຂື້ນເມື່ອປີ 1980 ເມື່ອ Tim Berners Lee ສະເໜີຕົ້ນແບບສໍາລັບນັກວິໃຈໃນ CERN ເພື່ອແລກປ່ຽນເອກະສານ ຂໍ້ມູນດ້ານການວິໃຈໂດຍໃຊ້ຊື່ Enquire ໃນປີ 1990 ເພີ່ນໄດ້ຂຽນໂປຣແກມບາວເຊີ ແລະ ທົດລອງລັນເທີງເຊີເວີທີ່ເພີ່ນພັດທະນາຂື້ນ HTML ໄດ້ຮູ້ຈັກຈາກ HTML Tag ເຊີ່ງມີຢູ່ 18 Tag ໃນປີ 1991 HTML ຖືກພັດທະນາຈາກ SGML ຢູ່ໃນຕອນນັັ້ນ ຕໍ່ມາໃນປີ 1996 ເພື່ອກໍານົດມາດຕະຖານໃຫ້ຕົງກັນ W3C World Wide Web Consortium ຈື່ງເປັນຜູ້ກໍານົດສະເປັກທັງໝົດຂອງ HTML ແລະ ປີ 1999 HTML 4.01 ກໍຖືກໍາເນີດຂື້ນ ໂດຍມີ HTML 5 ເຊີ່ງເປັນ Web Hypertext Application ຖືກພັດທະນາຕໍ່ມາໃນປີ 2004 ນອກຈາກນີ້ຍັງມີການພັດທະນາໄປເປັນ XHTML ເຊີ່ງຄື Extended HTML ເຊີ່ງມີຄວາມສາມາດ ແລະ ມາດຕະຖານທີ່ເຂັ້ມງວດກວ່າອີກດ້ວຍ ໂດຍພາຍໃຕ້ການຄຸມຄອງຂອງ W3C (World Wide Web Consortium).
